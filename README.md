# Aplikasi DermaKita (Santunan Anak Yatim)

 Deskripsi Singkat

Aplikasi **DermaKita** adalah proyek Frontend yang dikembangkan menggunakan **Node.js, React, Vite, dan TailwindCSS**. Aplikasi ini dibuat sebagai bagian dari laporan magang dan berfokus pada simulasi sistem santunan untuk anak yatim dengan tiga peran utama:

  Admin
  elawan
  User

Proyek ini masih berbasis data dummy dan bertujuan untuk mendemonstrasikan tampilan antarmuka serta alur interaksi pengguna.

---

 Cara Menjalankan Project

1. Clone repository ini:

   ```bash
   git clone https://github.com/diantikalyana/Aplikasi-DermaKita-Santunan-Anak-Yatim.git
   ```
2. Masuk ke folder project:

   ```bash
   cd Aplikasi-DermaKita-Santunan-Anak-Yatim
3. Install dependencies:

   ```bash
   npm install
   ```
4. Jalankan project dengan Vite:

   ```bash
   npm run dev
   ```
5. Buka browser dan akses:

   ```
   http://localhost:5173
   ```

 Catatan

* Proyek ini hanya mencakup Frontend.
* Data yang ditampilkan masih berupa dummy data.
* Tujuan utamanya untuk mendemonstrasikan tampilan aplikasi serta alur penggunaan tiga role utama.

---

Tentang Aplikasi
Aplikasi ini dirancang untuk memudahkan proses **donasi dan santunan kepada anak yatim**. Dengan adanya 3 role berbeda, setiap pengguna dapat mencoba perspektif yang berbeda:
Admin: mengelola data dan pengguna serta memverifikasi data baru.
Relawan: membantu distribusi dan penasarufan.
User: melakukan donasi dan melihat progres serta mengajukan data baru.
